# Soner Aledzhikov Portfolio

A personal front-end developer portfolio website built with React, Vite and React Router.

## Live Demo

Coming soon.

## About

This is my personal portfolio website.  
It presents my front-end projects, skills, learning progress and contact links.

The project was built as part of my front-end learning path and includes real deployed projects made with HTML, CSS, JavaScript and React.

## Features

- Home page with hero section
- Skills section
- Featured projects section
- Projects page with project cards
- About page
- Contact page
- Footer
- 404 page
- React Router navigation
- Responsive design
- GitHub and Live Demo links

## Technologies Used

- React
- Vite
- React Router
- JavaScript
- CSS
- Git
- GitHub
- GitHub Pages
